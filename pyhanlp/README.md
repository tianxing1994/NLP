### 安装: 
```text
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pyhanlp
```
说明: 
* 安装后, 第一次使用时, 会下载 hanlp 的数据文件, 这些文件比较大, 一般都会下载失败, 推荐手动(迅雷)下载并放到要求的路径下. 
* 手动下载的文件名和目标文件名可能不一样, 需要注意一下.
