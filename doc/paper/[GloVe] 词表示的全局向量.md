## GloVe 词表示的全局向量

https://nlp.stanford.edu/pubs/glove.pdf



### 摘要

用于学习单词的向量空间表示的最新方法已经成功地使用向量算术捕获了细粒度的语义和句法规则性, 但是这些规刚性的起源仍然是不透明的. 我们分析并明确了此类规则性在单词向量中出现所需的模型属性. 结果是一个新的全局对数线性回归模型, 该模型结合了文献中两个主要模型族的优点: 全局矩阵分解和局部上下文窗口方法. 我们的模型通过仅对单词-单词共现矩阵中的非零元素进行训练, 而不对整个稀疏矩阵或大型语料库中的单个上下文窗口进行训练, 从而有效地利用了统计信息. 该模型产生具有有意义子结构的向量空间, 这在最近的单词类比任务上表现为 75%. 它在相似性任务和命名实体识别方面也优于相关模型. 



### 1. 介绍

语言的语义向量空间模型用实值向量表示每个单词. 这些向量可用作各种应用程序的特征, 例如信息检索, 文档分类, 问题解答, 命名实体识别, 和解析. 

大多数单词向量方法都依赖于单词向量对之间的距离或角度, 作为评估此类单词表示形式的内在质量的主要方法. 最近, Mikolov 等. 提出了一种新的基于词类比的评估方案, 该方案通过检查词向量之间的标量距离, 而不是词向量之间的不同维数, 来探究词向量空间的更精细结构. 例如, 类比 "king is to queen as man is to woman" 在向量空间中应通过矢量方程式 $king - queen = man - woman$. 这种评估方案偏向于产生意义维度的模型, 从而捕捉分布式表示的多聚类思想. 用于学习单词向量的两个主要模型族是: 1. 全局矩阵分解方法, 例如潜在语义分析(LSA). 2. 局部上下文窗口方法, 例如 Skip-gram 语言模型, Mikolov 等. 目前, 两个方法都遭受重大不利因素. 虽然像 LSA 这样的方法有效地利用了统计信息, 但它们在词类比任务上的表现相对较差, 表明次优向量空间结构. 像 skip-gram 这样的方法在类比任务上可能会做得更好, 但是由于它们在单独的局部上下文窗口而不是在全局共现计数上进行训练, 因此它们无法充分利用语料库的统计信息. 

在这项工作中, 我们分析了产生含义的线性方向所必需的模型属性, 并认为全局对数-双线性回归模型适合这样做. 我们提出了一个特定的加权最小二乘模型, 该模可对全局单词-单词共现计数进行训练, 从而有效地利用统计信息. 该模型产生具有有意义子结构的词向量空间, 这在词类比数据集上具有 75% 的准确度. 我们还证明, 在几个词相似性任务以及通用命名实体识别 (NER) 基准上, 我们的方法优于其他当前方法. 我们在以下位置提供了模型的源代码以及经过训练的单词向量: http://nlp.stanford.edu/projects/glove/. 



### 2. 相关工作 

**矩阵分解方法**. 用于生成低维单词表示的矩阵分解方法的根源可以追溯到 LSA. 这些方法利用低秩近似来分解捕获有关语料库的统计信息的大型矩阵. 由此类矩阵捕获的信息的特定类型因应用程序而异. 在 LSA 中, 矩阵是 "术语文档" 类型的, 即行对应于单词或术语, 而列对应于语料库中的不同文档. 相反, 例如, 超空间语言模拟 (HAL) 使用 "术语项" 类型的矩阵, 即, 行和列对应于单词, 条目对应于给定单词出现在另一个给定单词的上下文中的次数. 

HAL 与相关方法的主要问题是, 最频繁出现的单词对相似性度量的贡献不成比例: 两个单词与 or 和 and 共同出现的次数, 例如, 尽管传达的语义相关性相对较少, 但对它们的相似性影响很大. 存在许多解决 HAL 缺点的技术, 例如 COALS 方法, 其中共现矩阵首先通过基于熵或相关性的归一化进行转换. 这种类型转换的优点是原始共现计数 (对于合理大小的语料库可能跨越 8 或 9 个数量级) 被压缩, 以便在较小的间隔中更均匀地分布. 各种更新的模型也采用这种方法, 包括一项研究表明正点向互信息是一种很好的转变. 最近, 有人提出以 Hellinger PCA (HPCA) 形式进行平方根类型转换是学习单词表示的有效方法. 

**简单的基于窗口的方法**. 另一种方法是学习单词表示法, 以帮助在本地上下文窗口内进行预测. 例如, Bengio 等. 引入了一种学习单词向量表示的模型, 该模型是用于语言建模的简单神经网络架构的一部分. Collobert 和 Weston 将单词向量训练与下游训练目标脱钩, 这为 Collobert 等人铺平了道路. 使用单词的完整上下文来学习单词表示, 而不是像语言模型那样仅使用前面的上下文.  

最近, 人们已经开始质疑完整的神经网络结构对于学习有用的单词表示的重要性. Mikolov 等人的 skip-gram 和 CBOW 模型. 提出了一种基于两个词向量之间的内积的简单单层架构. Mnih 和 Kavukcuoglu 也提出了密切相关的矢量对数双线性模型, vLBL 和 ivLBL, 以及 Levy 等人. 提出了一种基于 PPMI 度量的显式词嵌入. 

在 skip-gram 和 ivLBL 模型中, 目标是在给定单词本身的情况下预测单词的上下文, 而 CBOW 和 vLBL 模型中的目标是在给定单词上下文的情况下预测单词. 通过对单词类比任务的评估, 这些模型证明了学习语言模式作为单词向量之间线性关系的能力. 与矩阵分解方法不同, 基于简单窗口的方法具有以下缺点: 它们不能直接对语料库的共现统计进行操作. 相反, 这些模型扫描整个语料库的上下文窗口, 这无法利用模式在大量数据中的重复. 



### 3. GloVe 模型

语料库中单词出现的统计信息是所有用于学习单词表示法的无监督方法可用的主要信息来源, 尽管现在存在许多此类方法, 但问题仍然在于如何从这些统计信息中产生含义, 以及结果词向量如何表示这种含意. 在本节中, 我们将阐明这个问题. 我们使用洞察力为全局向量构建一个新的词表示模型, 我们称其为 GloVe, 因为该模型直接捕获了全局语料库统计信息. 

首先, 我们建立一些符号. 用 $X$ 表示单词-单词共现计数的矩阵. 其条目 $X_{ij}$ 表示单词 $j$ 在单词 $i$ 的上下文中出现的次数. 令 $X_{i} = \sum_{k}{X_{ik}}$ 表示任意单词在单词 $i$ 的上下文中出现的次数. 最后, 令 $P_{ij} = P(j \vline i) = X_{ij} / X_{i}$ 表示单词 $j$ 在单词 $i$ 的上下文中出现的概率. 

我们从一个简单的标例如始, 该示例展示了如何直接从共现概率中提取某些特定意义. 考虑表现出特定兴趣的两个词 $i$ 和 $j$. 对于具体性, 假设我们对热力学阶段的概念感兴趣, 为此我们可以令 $i = \text{冰} $, $j = \text{蒸汽} $. 这些单词之间的关系可以通过研究它们与各种探测单词 $k$ 的同现概率之比来检查. 对于与冰相关但与蒸汽无关的单词 $k$, 例如 $k = \text{solid}$, 我们猜测比率 $P_{ik} / P_{jk}$ 将很大. 类似地, 对于与蒸汽但与冰无关的单词 $k$ 例如 ( $k = \text{gas}$), 这个比率将会较小. 对于与水和冰都相关, 或与两者都无关的像 $\text{water}$ 或 $\text{fashion}$ 之类的词 $k$, 比率应接近 $1$. 表 1 显示了这些概率及其在大型语料库中的比率, 这些数字证实了这些期望. 与原概率相比, 比率可以更好地区分相关单词 ($\text{solid}$ 和 $\text{gas}$) 和不相关单词 ($\text{water}$ 和 $\text{fashion} $), 也可以更好地区分两个相关单词. 

上面的论点表明, 单词向量学习的适当起点应该以共现概率的比率而不是概率本身为准. 注意到比率 $P_{ik} / P_{jk}$ 取决于三个词 $i$, $j$ 和 $k$, 最一般的模型采用以下形式: 

$$\begin{aligned} F(w_{i}, w_{j}, \widetilde{w}_{k}) = \frac{P_{ik}}{P_{jk}} \end{aligned}, $$ 

其中: 

* $w \in \mathbb{R}^{d}$ 是 $d$ 维词向量. 
* $\widetilde{w} \in \mathbb{R}^{d}$ 是分离上下文词向量, 它的角色将在第 4.2 节讨论. 

在此等式中, 右侧是从语料库中提取的, 并且 $F$ 可能取决于某些尚未确定的参数. $F$ 的可能性很大. 但通过执行一些需求, 我们可以选择一个独特的选择. 首先, 我们希望 $F$ 在单词向量空间中以比率 $P_{ik} / P_{jk}$ 对信息进行编码. 由于向量空间本质上是线性结构, 因此最自然的方法是利用向量差. 为了这个目的, 我们可以将考虑仅限于仅依赖两个目标词的差异的那些函数 $F$, 从而修改等式为: 

$$\begin{aligned} F(w_{i} - w_{j}, \widetilde{w}_{k}) = \frac{P_{ik}}{P_{jk}} \end{aligned}, $$ 

接下来, 我们注意到, 上式中 $F$ 的参数. 是一个向量, 而右侧是标量. 虽然 $F$ 可以被认为是由例如神经网络参数化的复杂函数, 但这样做会使我们试图捕获的线性结构变得模糊. 为了避免这个问题, 我们可以先取参数的点积, 

$$\begin{aligned} F((w_{i} - w_{j})^{T}\widetilde{w}_{k}) = \frac{P_{ik}}{P_{jk}} \end{aligned}, $$ 

这可以防止 $F$ 以不希望的方式混合矢量维度. 接下来, 注意到, 对于单词-单词共现矩阵, 单词和上下文单词之间的区别是任意的, 并且我们可以自由交换这两个角色.  即: $P_{ik} = P_{ki}$, $P_{jk} = P_{kj}$. 由于向量的内积也是与顺序无关的. 所以可以想到令 $F(w_{i}^{T} \widetilde{w}_{k}) = P_{ik} = P_{ki}$, $F(w_{j}^{T} \widetilde{w}_{k}) = P_{jk} = P_{kj}$. 则有: 

$$\begin{aligned} \frac{F^{'}(w_{i}^{T} \widetilde{w}_{k})}{F^{'}(w_{j}^{T} \widetilde{w}_{k})} = \frac{P_{ik}}{P_{jk}} \end{aligned}, $$ 

为了令向量相减, 与向量内积的性质统一起来: 

$$\begin{aligned} F((w_{i} - w_{j})^{T}\widetilde{w}_{k}) = F(w_{i}^{T}\widetilde{w}_{k} - w_{j}^{T}\widetilde{w}_{k}) = \frac{F^{'}(w_{i}^{T} \widetilde{w}_{k})}{F^{'}(w_{j}^{T} \widetilde{w}_{k})} = \frac{P_{ik}}{P_{jk}} \end{aligned}, $$ 



容易想到 $F$ 是指数函数 $\exp$. 

则有: 

$$\begin{aligned} F((w_{i} - w_{j})^{T}\widetilde{w}_{k}) = F(w_{i}^{T}\widetilde{w}_{k} - w_{j}^{T}\widetilde{w}_{k}) = \exp{(w_{i}^{T}\widetilde{w}_{k} - w_{j}^{T}\widetilde{w}_{k})} = \frac{\exp(w_{i}^{T} \widetilde{w}_{k})}{\exp(w_{j}^{T} \widetilde{w}_{k})} = \frac{P_{ik}}{P_{jk}} \end{aligned}, $$ 

令: 

$$\begin{aligned} \exp(w_{i}^{T} \widetilde{w}_{k}) = P_{ik} = \frac{X_{ik}}{X_{i}} \\ \exp(w_{j}^{T} \widetilde{w}_{k}) = P_{jk} = \frac{X_{jk}}{X_{j}} \end{aligned}$$ 

其中: 

* $X_{ik}$ 是单词 $k$ 在单词 $i$ 的上下文中出现的次数. 
* $X_{i}$ 是单词 $i$ 出现的次数. 



则有: 

$$\begin{aligned} w_{i}^{T} \widetilde{w}_{k} = \log{(\frac{X_{ik}}{X_{i}})} = \log{X_{ik}} - \log{X_{i}} \end{aligned}$$ 

接下来, 注意上式中, 如果没有 $\log{X_{i}}$ 项, 将表现出 $i$, $k$ 的交换对称性. 但是, 该项与 $k$ 无关, 因此可以将其吸收到 $w_{i}$ 的偏置 $b_{i}$ 中. 最后, 为 $\widetilde{w}_{k}$ 增加一个额外的偏置项 $\widetilde{b}_{k}$, 

$$\begin{aligned} w_{i}^{T} \widetilde{w}_{k} + b_{i} + \widetilde{b}_{k} = \log{X_{ik}} \end{aligned}$$ 

上式是一个极大简化的等式. 但实际上是一个定义不清的等式, 因为 log 对数函数的参数不能为 0. 解决此问题的一种方法是在对数中包括加法运算, $\log{(X_{ik})} \rightarrow \log{(1 + X_{ik})}$, 其中保持 $X$ 矩阵的稀疏性, 同时避免差异. 将共现矩阵的对数分解的想法与 LSA 密切相关, 我们将在实验中使用所得模型作为基线. 该模型的主要缺点是, 它会同等权衡所有同时发生的事件, 甚至很少或永远不会发生的事件. 这种罕见的共现情况是噪声, 它比频繁出现的共现情况带来更少的信息, 但是即使是零输入也占 $X$ 数据的 $75-95%$, 具体取决于词汇量和语料库. 我们提出了一个新的加权最小二乘回归模来解决这些问题. 将前面的等式用于最小二乘问题, 并引入权重函数 $f(X_{ij})$ 到损失函数中, 我们得到模型: 

$$\begin{aligned} J = \sum_{i,j=1}^{V}{f(X_{ij})(w_{i}^{T} \widetilde{w}_{j} + b_{i} + \widetilde{b}_{j} - \log{X_{ij}})^{2}} \quad (8)\end{aligned},$$ 

其中: $V$ 是词汇量大小. 权重函数应遵循以下性质: 

1. $f(0) = 0$. 如果 $f$ 是一个连续函数, 当 $x \rightarrow 0$时, 它应下降得足够快以使得 $\lim_{x \rightarrow 0}{f(x) \log^{2}{x}}$ 的值是为限的. 
2. $f(x)$ 应该是非减函数, 以使得罕见的共现情况不会被过分加权. 
3. 对于较大的 $x$ 值, $f(x)$ 应相对较少, 以使得频繁出现的共现情况不会被过分加权. 

当然, 有很多函数可以满足这样的性质, 但是我们发现可以很好地工作的一类函数可以被参数化为: 

$$f(x) = \left\{ \begin{aligned} (x / x_{max})^{\alpha} \quad & \text{if} \space x < x_{max} \\ 1 \quad & \text{otherwise} \end{aligned} \right. \quad (9)$$ 

模型的性能一定程度上取决于临界值. 在我们所有的实验中, 我们因定使用 $x_{max} = 100$. 我们发现 $\alpha = 3/4$ 比 $\alpha = 1$ 对于线性模型有适度的改进. 虽然我们仅提供选择 $3/4$ 的经验动机, 但有趣的是, 在 (Mikolvo et al., 2013a) 中发现类似的分数幂定标, 以提供最佳性能. 

 

#### 3.1 与其它模型的关系

因为所有用于学习词向量的无监督学习方法最终都基于语料库的发生统计, 所以模型之间应该有共同点. 尽管如此, 某些模型在这方面仍然有些不透明, 尤其是最近的基于窗口的方法, 例如 skip-gram 和 ivLBL. 因此, 在本小节中, 我们将说明这些模型与方程式中定义的我们提出的模型之间的关系. skip-gram 或 ivLBL 模型的起点是构建一个 $Q_{ij}$ 模型, 用于表示单词 $j$ 在单词 $i$ 的上下文中出现的概率. 为了具体起见, 我们假设 $Q_{ij}$ 是 $softmax$, 

$$\begin{aligned} Q_{ij} = \frac{\exp{(w_{i}^{T} \widetilde{w}_{j})}}{\sum_{k=1}^{V}{\exp{(w_{i}^{T} \widetilde{w}_{k})}}} \quad (10)\end{aligned}$$

这些模型的大多数细节与我们的目的无关, 除了以下事实外, 它们试图在上下文窗口扫描语料库时使对数概率最大化. 训练过程以在线, 随机的方式进行, 但是隐含的全局目标函数可以写成: 

$$\begin{aligned} J = - \sum_{i \in corpus \\ j \in context(i)}{\log Q_{ij}} \quad (11) \end{aligned}. $$ 

在此总和中评估每一项的 softmax 的归一化因子是耗时的. 为了进行有效的训练, skip-gram 和 ivLBL 模型向 $Q_{ij}$ 引入了近似值. 然而, 如果我们首先将对 $i$ 和 $j$ 具有相同值的那些项组合在一起, 则可以更有效地评估上式. 

$$\begin{aligned} J = - \sum_{i = 1}^{V} \sum_{j=1}^{V} {X_{ij} \log Q_{ij}} \quad (12) \end{aligned}. $$ 

在这里, 我们使用了一个事实, 即相似项的数量由共现矩阵 $X$ 给出. 回顾我们对于 $X_{i} = \sum_{k}{X_{ik}}$ 和 $P_{ij} = X_{ij} / X_{i}$ 的表示法, 我们可以将 $J$ 重写为: 

$$\begin{aligned} J = - \sum_{i = 1}^{V}X_{i} \sum_{j=1}^{V} {P_{ij} \log Q_{ij}} = \sum_{i=1}^{V}{X_{i} H(P_{i}, Q_{i})} \quad (13) \end{aligned}. $$ 

其中: $H(P_{i}, Q_{i})$ 是分布 $P_{i}$ 和 $Q_{i}$ 的交叉熵, 类似于 $X_{i}$ 的定义. 作为交叉熵误差的加权和, 此目标与等式 8 的加权最小二乘目标在形式上有些相似. 实际上, 可以直接优化方程式 13, 这与 skip-gram 和 ivLBL 模型中使用的在线训练方法相反. 可以将这一目标解释为 "全局 skip-gram" 模型, 进一步研究可能很有趣. 另一方面, 等式 13 表现出许多不良的特征, 在将其用作学习单词向量的模型之前应加以解决. 

首先, 交叉熵误差只是概率分布之间许多可能的距离度量中的一种, 它具有不好的性质, 即长尾分布通常建模较差, 对不可能发生的事件给予了过多的权重. 此外, 对于要限制的度量, 要求模型分布 Q 正确归一化. 由于等式 10 中整个词汇量的总和, 因此出现了计算瓶颈. 并且考虑一个不要求 Q 的此属性的不同距离度量. 自然选择是一个最小二乘目标, 其中 Q 和 P 中的归一化因子被丢弃. 

$$\begin{aligned} \hat{J} = \sum_{i, j}{X_{i} (\hat{P}_{ij} - \hat{Q}_{ij})^{2}} \quad (14) \end{aligned}. $$ 

其中: $\hat{P}_{ij} = X_{ij}$, $\hat{Q}_{ij} = \exp{(w^{T}_{i} \widetilde{w}_{j})}$ 是未正则化的分布. 在这个阶段, 出现了另一个问题, 即 $X_{ij}$ 通常取非常大的值, 这会使优化变得复杂. 一种有效的补救方法是最小化 $\hat{P}$ 和 $\hat{Q}$ 对数的平方误差, 

$$\begin{aligned} \hat{J} &= \sum_{i, j}{X_{i} (\log \hat{P}_{ij} - \log \hat{Q}_{ij})^{2}} \\ &= \sum_{i, j}{X_{i} (w^{T}_{i} \widetilde{w}_{j} - \log X_{ij})^{2}} \end{aligned} \quad (15). $$ 

最后, 我们观察到, 虽然权重因子 $X_{i}$ 是由 skip-gram 和 ivLBL 模型固有的在线训练方法确定的, 但绝不能保证它是最优的. 实际上, Mikolov 等人. 观察到, 可以通过过滤数据来提高性能, 从而降低频繁单词加权因子的有效值. 考虑到这一点, 我们引入了一个更通用的加权函数, 我们也可以随意使用它来取决于上下文词. 结果是: 

$$\begin{aligned} \hat{J} &= \sum_{i, j}{f(X_{ij}) (w^{T}_{i} \widetilde{w}_{j} - \log X_{ij})^{2}} \end{aligned} \quad (16). $$ 

这等效于我们先前提出损失函数等式 8. 



#### 3.2 模型复杂度

从等式 8 和加权函数 $f(X)$ 的显式形式可以看出. 模型的计算复杂度取决于矩阵 $X$ 中非零元素的数量. 由于此数字始终小于矩阵条目的总数, 因此模型的缩放比例不会比 $O(|V|^{2})$ 差. 乍一看, 这似乎比基于简单窗口的方法有了实质性的改进, 该方法随语料库大小 $|C|$ 缩放. 但是, 典型的词汇有成千上万个单词, 因此 $|V|^{2}$ 可以达到数千亿, 这实际上比大多数语料库要大得多. 因此, 重要的是确定是否可以对 $X$ 的非零元素数量设置更严格的界限. 为了对 $X$ 中非零元数的数量做出任何具体说明, 有必要对单词共现的分布进行一些假设. 特别地, 我们将假设单词 $i$ 与单词 $j$ 的共现数 $X_{ij}$ 可以建模为该单词对 $r_{ij}$ 的频率等级的幂律函数: 

$$\begin{aligned} X_{ij} = \frac{k}{(r_{ij})^{\alpha}} \end{aligned}. \quad (17)$$ 

语料库中的单词总数与共现矩阵 $X$ 的所有元素之和成比例. 

$$\begin{aligned} {\vline C \vline} \sim \sum_{ij}{X_{ij}} = \sum_{r = 1}^{\vline X \vline}{\frac{k}{r^{\alpha}}} = kH_{\vline X \vline, \alpha} \end{aligned}. \quad (18)$$ 

我们用广义谐波数 $H_{n,m}$ 重写了最后的求和. 求和的上限 $\vline X \vline$ 是最大频率等级, 与矩阵 $X$ 中非零元素的数量一致. 此数字也等于等式 17 中的 $r$ 值. 这使得 $X_{ij} \ge 1$, $\vline X \vline = k^{1 / \alpha}$. 因此我们可以重写等式 18 为: 

$$\begin{aligned} {\vline C \vline} \sim {\vline X \vline}^{\alpha} H_{\vline X \vline, \alpha} \end{aligned}. \quad (19)$$ 

当两个数字都很大时, 我们对 $|X|$ 与 $|C|$ 的关系感兴趣. 因此, 我们可以自由扩展等式的右侧, 在 $|X|$ 很大时. 为此, 我们使用广义谐波数的扩展 (Apostol, 1976), 

$$\begin{aligned} H_{x, s} = \frac{x^{1-s}}{1 - s} + \zeta(s) + O(X^{-S}) \quad if \space x \gt 0, s \ne 1 \end{aligned}. \quad (20)$$ 

给出: 

$$\begin{aligned} {\vline C \vline} \sim \frac{\vline X \vline}{1 - \alpha} + \zeta(\alpha) {\vline X \vline}^{\alpha} + O(1) \end{aligned}. \quad (21)$$ 

其中: $\zeta(s)$ 是 Riemann zeta 函数. 在 $X$ 很大的极限中, 只有等式 21 右边的两个项之一是相关的, 并且它是哪一项, 这取决于 $\alpha > 1$,  

$${\vline X \vline} = \left\{ \begin{aligned} O(\vline C \vline) \quad & \text{if} \space \alpha < 1, \\ O({\vline C \vline}^{1/a}) \quad & \text{if} \space \alpha > 1. \end{aligned} \right. \quad (22)$$ 

对于此文研究的语料库, 我们观察到 $X_{ij}$ 是等式 17 通过 $\alpha = 1.25$ 精心建模的. 在这种情况下, 我们有 ${\vline X \vline} = O({\vline C \vline}^{0.8})$. 因此, 我们得出的结论是, 模型的复杂度比最坏情况下的 $O(V^{2})$ 好得多, 并且实际上, 它的性能比像 $O({\vline C \vline})$ 那样基于在线窗口的方法要好.
