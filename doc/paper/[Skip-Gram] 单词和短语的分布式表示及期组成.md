## 单词和短语的分布式表示及期组成

https://papers.nips.cc/paper/5021-distributed-representations-of-words-and-phrases-and-their-compositionality.pdf



### 摘要

最近引入的连续 Skip-gram 模型是学习高质量的分布式矢量表示形式的有效方法, 该表示形式捕获了大量精确的句法和语义词关系. 在本文中, 我们提出几种扩展, 可以同时提高矢量的质量和训练速度, 通过对频繁出现的单词进行二次采样, 我们可以获得显着的加速, 并且还可以学习更多常南侧单词的表示形式. 我们还描述了一种简单的替代方法, 称为分层 softmax, 称为负采样. 

单词表示的固有局限性是它们对单词顺序的漠不关心以及它们无法表示惯用语. 例如, "Canada" 和 "Air" 的含义不能轻易组合以获得得 "Air Canada". 受此示例的启发, 我们提出了一种用于在文本中查找短语的简单方法, 并表明学习数百万个短语的良好矢量表示形式是可行的. 



### 1. 介绍

向量空间中单词的分布式表示可帮助学习算法通过对相似单词进行分组来在自然语言处理任务中实现更好的性能. 最早使用词表示法的方法可以追溯到 1986 年的 Rumelhart, Hinton 和 Williams 等人. 此想法已被成功应用于统计语言建模. 后续工作包括自动语音识别和机器翻译的应用, 以及各种 NLP 任务. 

最近, Mikolov 等. 介绍了 Skip-gram 模型, 这是一种从大量非结构化文本数据中学习单词的高质量矢量表示的有效方法. 与大多数以前使用的用于学习单词向量的神经网络结构不同, Skip-gram 模型的训练不涉及密集矩阵乘法. 这使得训练过程非常高效: 经过优化的单机实现可以一天训练超过 1000 亿个单词. 

使用神经网络计算的单词表示形式非常有趣, 因为所学习的向量显式地编码为许多语言规律性和模式. 令人惊讶的是, 许多这些模式可以表示为线性平移. 例如, 向量计算的结果 vec("Madrid") - vec("Spain") + vec("France") 比任何其他单词向量都更接近 vec("Paris"). 

在本文中, 我们介绍几种对原始 Skip-gram 模型的扩展. 我们表明, 训练过程中频繁单词的子采样可显著提高训度速度 (大约 2 倍 - 10 倍), 并提高了不频繁单词的表示的准确性. 此外, 我们提出了一种噪声对比估计 (NCE) 的简化变体, 用于训练 Skip-gram 模型, 与之前使用的复杂的分层 softmax 相比, 该算法可更快地训练频繁词并是供更好的矢量表示工作. 

单词表示受其无法表示惯用语的限制, 这些惯用语不是由单个单词组成的. 例如, "Boston Globe" 是报纸, 因此它不是 "Boston" 和 "Globe" 的自然组合. 因此, 使用向量表示整个短语可以使 Skip-gram 模型的表达能力大大提高. 旨在通过组合词向量来表示句子含义的其他技术 (例如递归自动编码器) 也将从使用词组向量而不是词向量中受益. 

从基于单词的模型到基于短语的模型的扩展相对简单. 首先, 我们使用数据驱动的方法识别大量的短语, 然后在训练过程中将这些短语视为单独的标记. 为了评估短语向量的质量, 我们开发了一个包含单词和短语的类比推理任务测试集. 我们的测试集中有一个典型的类比对是 "Montreal": "Montreal Canadiens"::"Toronto":"TorontoMaple Leafs". 如果以下接近表示成立, 则它被认为是正确的. vec(“Montreal Canadiens”) - vec(“Montreal”) + vec(“Toronto”) = vec(“TorontoMaple Leafs”).

最后, 我们描述了 Skip-gram 模型的另一个有趣的特性. 我们发现简单的向量加法通常可以产生有意义的结果. 例如, vec("Russia") + vec("river") 接近于 vec("Volga River"),  vec("Germany") + vec("capital") 接近于 vec("Berlin"). 这种构成表明, 可以通过对词向量表示形式使用基本的数学运算来获得对语言的理解程度. 





### 2. Skip-gram 模型

Skip-gram 模型的训练目标是找到可用于预测句子或文档中周围单词的单词表示形式. 更正式地, 给定一系列训练词 w1, w2, w3, ..., wt, Skip-gram 模型的目标是最大化平均 log 对数概率. 

$$\begin{aligned} \frac{1}{T} \sum_{t=1}^{T}{\sum_{-c \le j \le c, j \ne 0}{\log p(w_{t+j} \vline w_{t})}} \end{aligned}$$ 

其中 c 是训练上下文的大小 (可以是中心词 wt 的函数). c 越大, 产生的训练示例越多, 因此可以导致更高的准确性, 却会浪费训练时间. 

基本的 Skip-gram 公式使用 softmax 函数定义 $p(w_{t+j} \vline w_{t})$. 

$$\begin{aligned} p(w_{O} \vline w_{I}) = \frac{\exp({v^{'}_{w_{O}}}^{T} v_{w_{I}} )}{\sum_{w=1}^{W}{\exp({v^{'}_{w}}^{T} v_{w_{I}} )}} \end{aligned}$$ 

其中: 

* 其中 $v_{w}$ 和 $v^{'}_{w}$ 是 input 和 output 向量的表示, W 是词汇表中单词的数量.
* 这种公式是不切实际的, 因为计算 $∇ \log p(w_{O}|w_{I})$ 的成本与 W 成正比, 而 W 通常很大 ($10^{5} - 10^{7} $). 





### 2.1 层次 Softmax

对完整 softmax 计算的完全有效近似计算是分层 softmax. 在神经网络语言模的背景下, 它是由 Morin 和 Bengio 首次提出的. 它的主要优点是, 无需评估神经网络中的 W 个输出节点以获得概率分布, 而是仅需要评估 $\log_{2}(W)$ 个节点. 

分层 softmax 使用 W 个词作为二叉树的叶子节点的表示形式, 并且对于每个节点, 显示表示其子节点的相对概率. 这些定义了随机游走, 将概率分配给单词. 

更精确地, 每个词 w 可以通过从树的根部开始的适当路径到达. 令 $n(w, j)$ 是从根到 w 的路径上的第 j 个节点, 令 $L(w)$ 为该路径的长度, 因此 $n(w, 1)=root$, $n(w, L(w)) = w$. 另外, 对于任何内部节点 n, 令 $ch(n)$ 为 n 的任意固定子项, 如果 x 为 true, 则令 [[x]] 为 1, 否则为 -1. 如此, 层次 softmax 定义 $p(w_{O} \vline w_{I})$ 如下: 

$$\begin{aligned} p(w \vline w_{I}) = \Pi_{j=1}^{L(w)-1}{\sigma ([n(w, j + 1) = ch(n(w, j))] \cdot {v^{'}_{n(w,j)}}^{T} v_{w_{I}})} \end{aligned}$$ 

其中: $\sigma(x) = 1 / (1 + \exp(-x))$. 可以验证 $\sum_{w=1}^{W}{p(w \vline w_{I})} = 1$. 这意味着计算 $\log p(w_{O} \vline w_{I})$ 和 $∇ \log p(w_{O} \vline w_{I})$ 的计算量正比于 $L(w_{O})$, 其的平均计算量不大于 $\log W$.  同样, 不同于一般的 softmax 函数给每一个词 w 一词对于一个 $v_{w}$ 和 $v^{'}_{w}$ 表示, 层次 softmax 给每个词 w 一个 $v_{w}$ 表示, 每一个二叉树节点一个 $v^{'}_{n}$ 表示. 

层次 softmax 的树结构在性能上具有非常好的效果. Mnih 和 Hinton 探索了许多方法来构造树结构, 以及对训练时间和模型精度的影响. 在我们的工作中, 我们使用了二叉霍夫曼树, 因为它为频繁的单词分配了短代码, 从而可以进行快速训练. 之前已经观察到, 通过频率将单词分组在一起, 对于基于神经网络的语言模型来说, 是一种非常简单的加速技术. 



### 2.2 负采样

分层 softmax 的替代方法是噪声对比估计 (NCE). 它由 Gutmann 和 Hyvarinen 提出, 并由 Mnih 和 Teh 应用于语言建模. NCE 认为, 好的模型应该能够通过逻辑回归将数据与噪声区分开. 这类似于 Collorbert 和 Weston 所使用的铰链损失, 他们通过将数据用于噪声之上对模型进行训练. 

尽管可以证明 NCE 使 softmax 的对数概率近似最大化, 但是 Skip-gram 模型仅关注学习高质量矢量表示, 因此只要矢量表示保持其质量, 我们就可以自由简化 NCE. 我们通过目标函数定义负采样 (NEG). 

$$\begin{aligned} \log \sigma ({v^{'}_{w_{O}}}^{T} v_{w_{I}}) + \sum_{i=1}^{k}{E_{w_{i} \sim P_{n}(w)}[\log \sigma (- {v^{'}_{w_{i}}}^{T} v_{w_{I}})]} \end{aligned}$$ 

它用于替代 Skip-gram 目标函数中的每一个 $\log P(w_{O} \vline w_{I})$ 项. 因此, 任务是使用逻辑回归从噪声分布 $P_{n}(w)$ 的绘图中区分出目标单词 $w_{O}$, 其中每个数据样本有 k 个负样本. 我们的实验表明, k 值在 5-20 之间对于小型训练数据集很有用, 而对于大型数据集, k 可以小到 2-5. 负采样与 NCE 之间的主要区别在于 NCE 既需要样本, 也需要噪声分布的数值概率, 而负采样仅使用样本. 虽然 NCE 近似最大化了 softmax 的对数概率, 但是此属性对于我们的应用并不重要. 

NCE 和 NEG 都将噪声分布 $P_{n}(w)$ 作为自由参数. 我们研究了 $P_{n}(w)$ 的多种选择, 发现提高到 3/4 次幂的 unigram 分布 $U(w)$ (如: $U(w)^{3/4} / Z$) 明显优于 unigram 和均匀分布, 对于 NCE 和 NEG, 我们尝试了包括语言建模在内的每项任务. 



### 2.3 根据词频负采样

在大型语料库中, 最常见的单词很容易出现数亿次. 这些词通常比稀有词提供更少的信息价值. 例如, 虽然 Skip-gram 模型受益于 "France" 和 "Paris" 的共现情况, 但它很少受益于 "France" 和 "the" 的共现, 因为几乎每个单词都经常与 "the" 共现. 这个想法也可以反方向应用. 在训练了数百万个示例后, 常用词的向量表示形式不会发生明显变化. 

为了消除稀有词和常见词之间的不平衡, 我们使用了一种简单的子采样方法: 训练集中的每个词 $w_{i}$ 均以公式计算的概率丢弃, 其中 $f(w_{i})$ 是单词 $w_{i}$ 的频率, t 是选定的阈值, 通常在 10-5 之间. 

$$\begin{aligned} P(w_{i}) = 1 - \sqrt{\frac{t}{f(w_{i})}} \end{aligned}$$ 

我们之所以选择这种二次采样公式, 是因为它在保持频率排名的同时, 主动对频率大于 t 的单词进行二次采样. 尽管此启发式公式是通过启发式方式选择的, 但我们发现它在实践中效果很好. 如以下各节所示, 它可以加快学习速度, 甚至可以显著提高衡有单词的学习向量的准确性.
